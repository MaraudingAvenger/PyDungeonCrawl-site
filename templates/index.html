PyDungeonCrawl

A feature-rich API for battling villains

Meet the Heroes

Get to know the heroes, their abilities, and their weaknesses. Skilled players will fully utilize hero synergies to defeat bosses.

Observe the Villains

Get to know the monsters you will face on your journey. Skilled players will take villains' skills into account, choosing heroes to counter or nullify the villains' strengths.

API Reference

Partial API reference with descriptions and usage examples of the relevant properties and methods you can use on your quest.

NOTE: This is not a complete API reference, but contains everything you should reasonably use/interact with as a player.

PyDungeonCrawl is best enjoyed with IPython/Jupyter (lab, notebook), although it's entirely useable on the command line or REPL. By default, the board is blitted to the display every quarter second. On some (windows) machines, this can cause a discomfiting flickering effect.
