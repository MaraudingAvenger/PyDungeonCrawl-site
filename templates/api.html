Installation

To install pydungeoncrawl: 

pip install pydungeoncrawl

Structure

Below is a small code template to get you started using the PyDungeonCrawl library.

In general, everything begins with a level. If you think of pydungeoncrawl as a board game, the level represents the board that the game is played on. Select a level from the pydungeoncrawl.levels module.

Select a boss to battle against from the pydungeoncrawl.bosses module. (See the Villains section for more information)

Select a party of heroes consisting of one tank, one healer, and two damage dealers. (See the Heroes section for more information) 


# import a level from pydungeoncrawl.levels
from pydungeoncrawl.levels import MovementTraining

# import a boss from pydungeoncrawl.bosses
from pydungeoncrawl.bosses import Golem

# import heroes and the party class from pydungeoncrawl.heroes
from pydungeoncrawl.heroes import Shaman, Paladin, Ranger, Bard, Party

# instantiate heroes and the boss
shaman = Shaman("Seamus")
paladin = Paladin("Perry")
ranger = Ranger("Ruprect")
bard = Bard("Barry")

boss = Golem()

# instantiate a party using the heroes
party = Party(shaman, paladin, ranger, bard)

# instantiate a level using the party and boss
game = MovementTraining(party=party, boss=boss)

for turn in game:
    # put all your logic here
    ...

Pawn (Character and Boss base class)

Properties

Pawn built in properties
Property Type Description
pawn.name str The name of the pawn. Example: 
hero = Paladin('Perry')
hero.name # 'Perry'
pawn.position tuple[int, int] The position of the pawn on the board. Example: 
hero = Paladin('Perry')
hero.position # Point(0, 0)
pawn.active_effects list[str] All current active effects on the pawn in the form ['effect 1', 'effect 2', ...]. Example: 
hero.active_effects # [ 'might', 'blind', 'poison']
pawn.blinded pawn.poisoned pawn.rooted bool True or False depending on if the pawn is currently being affected by the corresponding effect. Example: 
hero.blinded  # False
boss.poisoned # True
pawn.cooldowns dict[str, int] A dictionary of all current cooldowns on the pawn in the form {'cooldown 1': 0, 'cooldown 2': 3, ...}. Example: 
hero.cooldowns # {'smite': 1, 'divine shield': 3}
pawn.health int The current health of the pawn. Example: 
hero.health # 120
pawn.health_percent int The health of the pawn as a percentage of pawn.health_max. Example: 
hero.health_percent # 0.43
pawn.is_alive bool Whether or not the pawn is alive. Example: 
hero.is_alive # True
pawn.points_behind list[Point] Gets the three points that make up what is "behind" the character. This is useful for determining if a character is behind another character, or if a character is in range of an ability. pawn.is_behind() is a convenience function that uses this property. Example: 
hero.points_behind # [Point(5, 5), Point(5, 6), Point(5, 7)]
pawn.vulnerabilities pawn.resistances list[Effect] The pawn's vulnerabilities and resistances. These are the effects that the pawn is vulnerable to or resistant to, respectively. Example: 
boss.vulnerabilities # [Effect("Poison Vulnerability"), Effect("Magic Vulnerability"),...]

Boss

Most things available on Pawn, with the addition of:
Boss interface descriptions.
Entry Arguments Returns Description
repr(boss) None list[str] The repr for a Boss displays the name, class, and health of the boss. Example: 
from pydungeoncrawl.bosses import Golem
boss = Golem()
repr(boss) # "Thunk (Golem), 20000/20000 HP"
boss.telegraph None str A message indicating some upcoming move or action that the boss will take. If the boss is telegraphing something, chances are that it's important to the survival of the party.

Square

A single play space of a Board (When creating a Game object that includes a Board, the board is stored in the game.board property)

Properties

A description of the properties of the Square class
Entry Returns Description
square.position
 Point the position at which the Square is located. Example: 
board.at( (5, 5) ).position # Point(5, 5)
square.symbol str The string representation of the square when the board is displayed. Example: 
board.at( (9, 9) ).symbol # '🟥'
Note: the symbol will become the occupant's symbol if the square is occupied.
square.impassable bool Whether or not the Square is impassable due to being made of something impassable (wall, void) or because it is occupied by someone. Example: 
board.at( (5, 5) ).impassable # False
square.occupied bool Whether or not the Square is occupied by someone. Example: 
board.at( (7, 3) ).occupied # True
square.occupant Pawn|None The occupant of the Square if it is occupied, else None. Example: 
board.at( (7, 3) ).occupant # Character(Perry, (7, 3), 120/120, ⚜️)
square.damage int The amount of damage that the Square deals to anyone who steps on it. Example: 
board.at( (9, 9) ).damage # 10000
Note: This value only relevant if the Square is lava or burning.
square.is_burning bool Whether or not the Square is on fire. Example: 
board.at( (7, 3) ).is_burning # False
square.is_lava bool Whether or not the Square is lava. Example: 
board.at( (9, 9) ).is_lava # True

Methods

A description of the board API
Entry Arguments Returns Description
board.at
 Point|tuple[int,int] Square|None Returns the Square at the specified point. If there is no square at the specified point, returns None. Example: 
board.at(Point(5, 5)) # Square(Point(5, 5), "⬜")
board.at((5, 5)) # Square(Point(5, 5), "⬜")
board.at(Point(100, 100)) # None
board.get_adjacent_squares Point|tuple[int,int] list[Square] Returns a list of all the squares orthogonally or diagonally adjacent to the specified point. Example: 
game.board.get_adjacent_squares(Point(5, 5)) # [Square(Point(5, 4), "⬜"), Square(Point(5, 6), "⬜"),...]
board.get_adjacent_entities Point|tuple[int,int] list[Pawn] Returns a list of all the pawns orthogonally or diagonally adjacent to the specified point. Example: 
game.board.get_adjacent_entities((5, 5)) # [Character(Perry, (0, 0), 120/120, ⚜️),...]
board.get_squares_in_radius Point|tuple[int,int], int list[Square] Returns a list of all the squares within the specified radius of the specified point. Example: 
game.board.get_squares_in_radius((5, 5), 2) # [Square(Point(5, 4), "⬜"), Square(Point(5, 6), "⬜"),...]

Location Utilities

A collection of utilities found in the pydungeoncrawl.utilities.location module
A description of the board API
Entry Arguments Returns Description
class pydungeoncrawl.utilities.location.Point int, int Point A class representing a point in 2-dimensional space. Example: 
from pydungeoncrawl.utilities.location import Point
p = Point(4, 6)
p.x # 4
p.y # 6
pydungeoncrawl.utilities.location.distance_between Point|tuple[int,int], Point|tuple[int,int] float Returns the straight line distance between two points. Example: 
distance_between((5, 5), (10, 10)) # 7.0710678118654755
pydungeoncrawl.utilities.location.bresenham Point|tuple[int,int], Point|tuple[int,int] Generator[Point] Generate the integer points on a line between "origin" and "destination", not including "origin" using bresenham's line algorithm. Example: 
list(bresenham((5, 5), (8, 9))) # [(6, 6), (6, 7), (7, 8), (8, 9)]
Note: This is the algorithm used in the pawn.move_toward method. It is unaware of the board and does not take into account any obstacles or dangerous terrain; it simply generates a "pixelated" line between two points.
